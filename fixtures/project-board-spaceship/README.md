# Starfall Courier

Starfall Courier is a browser-based WebGL spaceship survival game. The player pilots a small courier ship through a dangerous asteroid lane while hostile drones try to intercept cargo.

The ideal experience is fast, readable, and replayable. It should be easy to launch locally, play within a few seconds, and understand whether the prototype works from a screenshot or short manual pass.

This fixture is intentionally incomplete. Use it to test whether Build Board can infer a useful project charter and ticketize several self-contained cards rather than creating one broad "make the game" task.

Known product direction:

- The first milestone should be a playable vertical slice in the browser.
- The prototype needs a visible ship, hazards, scoring, and a clear restart loop.
- Visual proof matters because a passing unit test does not prove the canvas is rendering a usable game.
- Keep early scope narrow; powerups, bosses, and campaign structure are not first-pass requirements.
